UN3 Exporter for Unity Asset Documentation
The UN3 Exporter is a Unity tool for exporting selected objects to the three.js format. Access the UN3 Exporter through the Unity menu by navigating to: Window > Ex Machina Soft > Export Selection to three.js.

Exporting Steps
Follow these steps to export your Unity objects to three.js:

Select Objects: In the Scene Hierarchy, choose the parent object or multiple objects that you wish to export.
Set Export Folder: Click the "Export Folder" button to specify the destination folder for the exported files.
Export: Click the "Export" button to initiate the export process. After a short wait, the tool will generate a three.js compatible JSON file. You can then open this file using the built-in ObjectLoader in three.js
